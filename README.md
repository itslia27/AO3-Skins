# AO3 Skins
Custom AO3 site skins made by me!!

## Hello!

This repository contains the CSS code for an AO3 skin I created.  
It is heavily inspired by *Constellations* by thevalkyrieflies on Tumblr.

To make customization easier, I’ve added comments in the code.  
AO3 automatically removes these comments when added to their CSS reader (lucky us!!)


## Customization Notes
Below are a few elements you may want to tweak to better suit your preferences.  
(Honestly, it's things I couldn't decide on)

### ✦ Relationship Tag Glow
You can adjust the glow effect by changing the last value in this (line 348):

```css
text-shadow: 0px 0px 3px var(--accent);
```
Increasing the last number will increase the glow, while the opposite is true for decreasing
